# Archivos auxiliares

Este directorio contiene un conjunto de archivos auxiliares necesarios para resolver el proyecto **Corrección de Iluminación** de la asignatura Visión Computacional.

Los archivos son los siguientes:

 * ```fitPlane.py```: Programa en python para ajustar datos 3D a un plano.
 * ```LibrosG.png```: Imagen Original.
 * ```Libros[1,2,3].png```: Imágenes con iluminación no uniforme.
 * ```Leeme.md```: Este archivo.
